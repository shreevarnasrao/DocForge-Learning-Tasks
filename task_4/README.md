**Task 4**


**This week, you'll be diving into Sequential Models — the backbone of how AI reads and understands text one step at a time.** You'll go from understanding why basic neural networks fall short on language tasks, to building RNNs, LSTMs, and GRUs in PyTorch from scratch. By the end, you'll have trained a small language model on actual Python function signatures — which is exactly the kind of sequence understanding that powers DocForge's documentation generation engine.
